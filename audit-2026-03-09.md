## 7. Bug Reproduction Steps

### BUG-001: lossless-claw not registered
1. Set `plugins.slots.contextEngine = "lossless-claw"` in openclaw.json
2. Restart gateway
3. Trigger any session operation
4. Check logs for "Context engine \"lossless-claw\" is not registered"

### BUG-002: micro-trigger-manager wrong session check
1. User sends message in current session
2. Wait 10 minutes for micro-trigger-manager to run
3. Check thinking-state.json lastUserMessage
4. Observe: timestamp not updated, micro-trigger starts incorrectly

### BUG-003: Tailscale serve failure
1. Restart gateway
2. Check logs immediately after startup
3. Observe: "tailscale serve failed: not running"

### BUG-004: 晚间复盘 cron error
1. Wait for 22:00 Beijing time
2. Check cron status: `openclaw cron list`
3. Observe: status = error

### BUG-005: API rate limits
1. Make multiple API calls in short time
2. Observe: "API rate limit reached" or "HTTP 402 quote_exceeded"
